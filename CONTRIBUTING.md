# Contributing

Thanks for considering contributing to this project!

## How to contribute

1. **Fork** the repository
2. **Create a feature branch** — `git checkout -b feature/my-feature`
3. **Make your changes**
4. **Run the bot locally** to verify it works — `python bot.py`
5. **Commit** with a clear message — `git commit -m "Add feature X"`
6. **Push** — `git push origin feature/my-feature`
7. **Open a pull request**

## Guidelines

- Keep changes focused — one feature/fix per PR
- Follow existing code style (the project is small, keep it readable)
- Update the README if adding or changing configuration
- Be respectful and constructive in discussions

## Questions?

Open an issue for any questions or ideas before starting work.
